As the Dean…

1.	I want a mobile app so that I can see/view projects/events occurring in all 9 of the departments in T&C.
2.	 I want to be able to add my input about the importance of events that have been submitted to me.
3.	 I want to be able to "zoom" in and out from daily, to weekly to monthly views to see everything headed my way at once, or just the most important items that day/week.
4.	 I want to be able to filter my views by department.
5.	I want editing/admin rights to edit the importance of events/remove events so that I can make necessary corrections to over-zealous/spamming Chairs.
6.	I want to be able to set reminders/notifications for events that I choose, so that my phone will remind me when/where I need to go.
7.	I want to be able to click/tap on a single event on the radar so that I can get a more detailed description of the event.
8.	 I want to receive a notification on my phone whenever a new event is added to the radar, so I can always know about upcoming events as soon as possible.
9.	i want to be able to add my own scale of importance to different projects/events according to how i see them.
10.	I would like to be able to add a project/event to my calendar.
11.	I want to see my app run on the web.
12.	I want to put my own input into the app.
13.	I want to access the app from anywhere including different devices (Desktop, tablets, or phone).
14.	 The app has to be user friendly and easy to use.
15.	 I want to be able to remove projects/events or have them auto-remove once the event date has expired.
16.	 I would like to receive e-mail reminders when a new project/event has been submitted.
17.	 I would like the radar app to export iCal invites so I can add them to my calendar. 
18.	I want an easy way to determine which event/project is deemed most important by me, and be able to categorize it that way.
19.	I would like to be able to see the events/projects in a list ordered by Importance, set by me, Date submitted, Date of event, 
20.	I would like to be able to add event/project dates directly to my calendar from the app.
21.	I would like to input feedback about items that are both private, for my eyes only, and that can be seen by the Chair Head that submitted it.
22.	I want to quickly see how soon appointments are and how important they are compared to others so that I can set my schedule accordingly.
23.	I want an app that visually shows me when my appointments are to quickly assess those that need my attention now.
24.	I would like a contact function, either phone or email, to easily respond to those setting my appointments in case I need to contact them about the appointment.
25.	I want to be able to see more information about each event so that I can access my own interest level
26.	 I want to be able delete events that I don't deem important so that I don’t have a lot of clutter on my screen
